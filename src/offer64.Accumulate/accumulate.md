关于如何递归实现a*b，：利用位运算来做，快速幂，快速模乘,

原理是把a拆成2的幂的和，a = 2^e0 + 2^e1 + 2^e2.... \
     那么 a * b = (2^e0 + 2^e1 + 2^e2+...) * b \
                      = b * 2^e0 + b * 2^e1 + b * 2^e2 + ...\
                      = (b << e0) + (b << e1) + ...
